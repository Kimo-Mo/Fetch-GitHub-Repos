# Fetch GitHub Repositories
Fetch GitHub Repositories is a simple project that allows users to input a GitHub username and retrieve a list of the user's repositories along with their links for both code access and visiting the repository.

## Features
 - Input a GitHub username to fetch the user's repositories.
 - Display the repository names and their links.
 - Links to visit the repository page.
 - Links to access the repository code.
 - Simple and user-friendly interface.

## Technologies Used
 - HTML
 - CSS
 - JavaScript
